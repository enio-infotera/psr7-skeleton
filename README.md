# PSR-7 and PSR-15 Hello world

Simple as possible PSR-7 / PSR-15 application (Hello World) with FastRoute and ErrorHandler middleware.

## Installation

```
git clone https://github.com/odan/psr7-hello-world.git .
composer update
```

## Starting

```
cd public/
php -S localhost:8080
```
